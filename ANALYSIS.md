## 3. 화면(UI) 분석 — 도구의 실제 기능
영상은 "인스타그램 자동 업로드 프로그램"이라는 **Windows 데스크톱 앱(WinForms 형태, 한글)** 의 화면을 보여줌.

- 상단: `id` / `password` 입력, **로그인 / 세팅 초기화 / 세팅 저장**
- 탭: **계정설정 · 내용설정 · 이미지설정 · 등록설정**
- 계정 등록행: `id` / `pw` / 계정 등록하기 / 계정 불러오기
- 계정 테이블 컬럼: 선택 · id · pw · **수량** · **딜레이** · 현황 · Progress
- 자막: "제목과 키워드와 특수문자를 조합해 자동 생성 / 키워드 중심의 자동 콘텐츠 생성"

## 4. 기능 종합 (레퍼런스 도구가 하는 일)
1. **다중 인스타 계정 관리** (계정별 id/pw)
2. **브라우저 자동화**로 instagram.com 웹 로그인 + 게시물 작성(비공식 자동화)
3. **콘텐츠 자동 생성**: 제목+키워드+특수문자 조합, 텍스트 리스트(순서/랜덤)
4. **이미지 처리 파이프라인**: 폴더 일괄 로드, 순서/랜덤, 글자 오버레이(테두리/그림자), 필터, 테두리, 리사이즈, 자동 다운로드
5. **사진 + 동영상** 업로드 지원
6. **수량·딜레이** 설정(탐지 회피 목적)

## 5. 컴플라이언스 판단 ⚠️
레퍼런스 도구는 **Instagram 약관 위반 + 스팸 성격**:
- 대량 계정 운영 + 비공식 브라우저 자동화 = ToS 위반, 계정 영구정지/법적 리스크
- 예시 콘텐츠가 명백한 광고 스팸

→ PRD에서는 **합법/약관준수 경로(공식 Graph API, 본인 소유 비즈니스 계정, 예약발행, 콘텐츠 템플릿/이미지 처리)** 를 핵심으로 설계하고, 대량계정·스팸 기능은 **명시적 비목표(Out of Scope)** 로 제외함.
