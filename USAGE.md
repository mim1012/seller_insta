# 사용법 — AI 이미지/영상 생성 & 관리

> 인스타그램 자동 포스팅 앱의 AI 미디어 생성 기능 사용법. 텍스트로 여러 장을 뽑아 → 갤러리에서 관리하며 → 일부만 골라 업로드 목록에 넣는 워크플로.

## 1. 프로바이더 선택 (OpenAI vs fal.ai)

`등록설정` 탭 → **이미지/영상 생성 프로바이더**에서 선택합니다.

| 프로바이더 | 특징 | 언제 |
|---|---|---|
| **OpenAI** | gpt-image-1 / sora-2, 고가 | 이미 OpenAI 키를 쓰고 있고 단가 신경 안 쓸 때 |
| **fal.ai 게이트웨이 (권장)** | 상세페이지/마케팅 특화 모델, 3~8배 저렴 | 대부분의 경우 |

> fal.ai를 고르면 fal API 키 입력란과 모델 선택이 나타납니다. **키는 https://fal.ai 에서 발급**받아 넣고 "fal 키 테스트"로 확인하세요.
> ⚠️ 캡션(글) 생성은 항상 OpenAI가 담당합니다. fal은 이미지/영상만 생성합니다. 글도 AI로 뽑으려면 OpenAI 키도 함께 넣으세요.

## 2. 특화 모델 (fal.ai 기본값)

앱이 자동으로 상황에 맞는 특화 모델을 씁니다.

| 용도 | 기본 모델 | 단가(약) |
|---|---|---|
| 상세페이지/제품 이미지 (텍스트→이미지) | **Seedream V4** (커머스 특화) | $0.03/장 |
| 제품 사진 배경교체 (이미지→이미지) | **Nano Banana edit** (아이덴티티 보존) | $0.039/장 |
| 마케팅 영상 (제품사진→광고컷) | **Seedance Pro** (제품광고 특화) | $0.62/5초 |
| 텍스트만으로 영상 | Kling 2.5 Turbo Pro | $0.35/5초 |

- 한글 문구가 들어간 이미지는 **AI에 맡기지 말고**, `이미지설정` 탭의 글자 오버레이(글자1/글자2·리스트)로 넣으세요. AI의 한글 렌더링은 불안정합니다. AI는 깔끔한 배경/제품 씬만 생성하게 하는 게 좋습니다.
- 단가는 벤더 정책상 수시로 바뀝니다. 실제 청구 전 각 모델 페이지에서 재확인하세요.

## 3. 텍스트로 여러 장 생성 → 골라 쓰기 (핵심)

`AI 생성` 탭에서:

1. **이미지 프롬프트**에 원하는 그림을 텍스트로 설명
   - 예: `화이트 배경 미니멀 향수병, 부드러운 자연광, 상단 여백 확보`
2. **생성 장수**를 지정 (기본 4장, 최대 10장)
3. **"여러 장 생성"** 클릭 → 후보들이 아래 **갤러리**에 썸네일로 표시됩니다
4. 마음에 드는 것만 **체크** (썸네일 클릭 또는 체크박스). `전체 선택` / `선택 해제` 버튼도 있음
5. **"선택한 항목 업로드 목록에 추가"** → 체크한 것만 실제 업로드 풀에 들어갑니다
6. 이후 `단일 발행` 또는 `배치 작업`에서 그 이미지들이 사용됩니다

> 여러 개 뽑아서 일부만 SNS에 쓰는 흐름이 이 갤러리로 완성됩니다.

## 4. 제품 사진으로 배경교체 / 광고영상 만들기

1. `AI 생성` 탭 → **"제품 사진 첨부(배경교체/영상용)"** 클릭 → 제품 사진 선택
2. 프롬프트에 원하는 결과 설명
   - 이미지 예: `배경을 밝은 스튜디오 화이트로 교체, 제품 형태·로고 유지`
   - 영상 예: `제품을 천천히 회전시키며 부드러운 조명, 9:16 세로`
3. 제품 사진이 첨부되면 앱이 자동으로 **배경교체(i2i) 또는 영상(i2v) 특화 모델**로 전환해 호출합니다
4. 결과를 갤러리/미리보기에서 확인 후 선택

## 5. 인스타그램 규격 팁

- 피드 정사각 **1:1 (1080×1080)**, 세로 **4:5 (1080×1350, 모바일 점유율 커서 선호)**, 릴스/스토리 **9:16 (1080×1920)**
- `AI 생성` 탭의 "이미지 크기"에서 목표 비율을 고르면 그 규격으로 생성됩니다

## 6. 라이선스

- **fal.ai 경유로 생성한 이미지/영상은 상업 이용 가능**합니다(FLUX 등 오픈웨이트 모델도 게이트웨이 경유 출력물은 허용).
- Google 계열(Nano Banana 등) 출력에는 비가시 워터마크(SynthID)가 들어갈 수 있으나 상업 배포엔 문제없습니다.

## 7. 준비물 요약

- **fal.ai API 키** (이미지/영상 생성용) — https://fal.ai
- (선택) **OpenAI API 키** (캡션 글 생성용) — https://platform.openai.com
- 키는 로컬에 OS 암호화(safeStorage)로 저장되며 평문으로 남지 않습니다.
